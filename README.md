# 暖禾农场

原创 2D 像素风农场小游戏基础版，无需安装依赖，直接在现代浏览器中运行。

## 启动

推荐在项目目录运行：

```bash
python3 -m http.server 8000
```

如果本机未配置 Python，也可以使用：

```bash
ruby -run -e httpd . -p 8000
```

然后访问 `http://localhost:8000`。

## 已实现

- 20×20 地图、木屋、商店、仓库
- WASD / 方向键移动与鼠标点击交互
- 锄头、水壶、镰刀、肥料、三种作物种子
- 翻地、播种、浇水、施肥、成长、成熟、收获
- 土豆（3 天）、胡萝卜（5 天）、草莓（8 天）
- 商店购买种子和肥料、出售收获物
- 背包、金币、体力、时间、日期和工具栏
- 睡眠与每日结算、10% 偷菜事件
- 本地自动存档

## 后续扩展建议
